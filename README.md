## Getting Started

### 1. Run the database

`docker compose up -d`

### 2. Create the .env file according to .env.example file

### 3. Prisma Commands

`npx prisma init`

`npx prisma migrate dev`

`npx prisma generate`

### 4. Execute seed

Api /api/seed

### 5. Run the development server

First, run the development server:

```
    npm run dev
```
